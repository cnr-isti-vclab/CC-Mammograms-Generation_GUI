The function from_model_to_512x512 converts the compressed breast model obtained by the GUI in a label with size 512x512. Please note that you will need to insert the path of the directory containing all the models into the code (line 2) and the path of the directory where the labels will be saved (line 3). The labels will be saved with the same names as the models to ensure the identifiers are preserved.

The function from_mammo_to_512x512 converts the real mammograms in images with size 512x512.
